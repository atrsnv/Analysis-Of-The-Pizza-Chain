# Analysis-Of-The-Pizza-Chain

![image](https://github.com/rezzstra/Analysis-Of-The-Pizza-Chain/assets/142921009/fa783a6d-b2e0-4dd2-8526-0ccad7d3d068)

#### В рамках данного проекта был проведен анализ сети пиццерий с целью выявления ключевых факторов, влияющих на успешность бизнеса. Исследование охватывало анализ данных о продажах, популярности отдельных видов пицц, эффективности маркетинговых кампаний.
